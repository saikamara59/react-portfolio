## Features:

- Modern Tech Stack: Built using Next.js for server side rendering and routing, Tailwind Css for utility-first styling and Framer Motion for sleek animations

- Responsive Design: Optimized for all devices, from mobile phones to desktops and tablets.

- Interactive UI: Smooth page transitions and scroll animations using Framer Motion.

- Project Showcase: Highlights my best work with links to live demos and Github repos.

- Contact Section: Easy way for visitiors to get in touch with me.


## Tech Stack

- Framerwork: Next.js

- Styling: Tailwind Css

- Animation: Framer Motion

- Deployment: Netlify

## Getting Started 
To run the project locally:

# Clone the repo
git clone https://github.com/saikamara59/react-portfolio.git

# Navigate to the project directory
cd react-portfolio

# Install dependencies
npm install

# Run the development server
npm run dev
